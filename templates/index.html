  1. 위에서 figma_plugin_code.js 파일을 다운로드합니다.
  2. Figma Desktop 앱을 실행합니다.
  3. 메뉴에서 Plugins → Development → Import plugin from manifest...를 클릭합니다. (최초 1회)
  4. antigravity-figma-plugin/manifest.json 파일을 선택합니다.
  5. 이후 Plugins → Development → Antigravity PSD Importer를 실행합니다.
  6. 플러그인 UI에 다운로드된 .js 파일을 드래그 앤 드롭합니다.
  7. ⚡ Figma에 임포트 버튼을 클릭하면 완료!
